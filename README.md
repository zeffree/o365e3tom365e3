# o365e3tom365e3

Interactive simulator that quantifies the **bundle savings** from upgrading from **Office 365 E3** to **Microsoft 365 E3** instead of buying Office 365 E3 plus separate 3rd-party identity, device-management, endpoint-protection, information-protection, and Windows Enterprise products.

## Concept
- **Option A — ala carte:** Office 365 E3 (\) + 3rd-party best-of-breed for each capability M365 E3 bundles in
- **Option B — bundle:** Microsoft 365 E3 (\) — single SKU covering Office 365 E3 + EMS E3 (Entra ID P1, Intune, MDE P1, AIP P1) + Windows 11 Enterprise

## Features
- Live cost computation for any user count, term (1/3/5 yr), and negotiated discount
- ~30 vendors curated from the latest **Gartner Magic Quadrants** (Access Management 2024, UEM Tools 2024, EPP 2025) tagged Leader / Challenger / Visionary / Niche Player
- Side-by-side line-item breakdown
- Annual cost chart + Option A spend-mix doughnut
- Capability parity matrix
- Light & dark themes (auto-detects OS preference, persisted)

## Use
Open `index.html` in any modern browser. No build, no server.
